快速体验

  1. 克隆当前项目到本地：

    git clone https://github.com/coldnight/dump-brain-with-emacs.git ~/dump-brain-with-emacs

  2. 确保 Emacs 已经安装，如果没有请先安装 Emacs 最新版。

  3. 使用当前项目 Emacs 配置文件启动项目：

    env HOME=~/dump-brain-with-emacs emacs

  4. 等待包管理自动安装相关插件。
